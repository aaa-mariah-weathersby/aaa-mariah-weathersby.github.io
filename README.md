## Public Website

http://autoclub-olb.s3-website-us-west-1.amazonaws.com/

The contents of this site is deployed via the aws cli.  'withAWS' jenkinsFile plugin and 's3Upload' are used for deployment. By setting the acl to "public-read", this allows the files in the bucket to be publically accessible.

```
aws s3 cp build s3://autoclub-olb --recursive --acl public-read
```

In order for the command above to run, the S3 Bucket's Policy needs to be set to the following

```
{
    "Version": "2012-10-17",
    "Statement": [
        {
            "Sid": "Stmt1528349026830",
            "Effect": "Allow",
            "Principal": "*",
            "Action": [
                "s3:PutObject"
            ],
            "Resource": "arn:aws:s3:::autoclub-olb/*"
        }
    ]
}
```
